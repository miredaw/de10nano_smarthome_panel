Temperature Sensor

BME280 (I²C, address 0x76)

OFFLINE
Current Reading
-- °C
Module:Waveshare BME280 Environmental Sensor
Interface:I²C via FPGA GPIO (JP1 Pins 1–2)
SDA / SCL:JP1 Pin 1 (PIN_V12) / JP1 Pin 2 (PIN_E8)
I²C Address:0x76 (ADDR pin → GND)
Pull-ups:4.7 kΩ on-module (no external resistors needed)
Power:3.3V from JP1 Pin 30 / GND from JP1 Pin 29
LW Bridge Base:0xFF200000 (bme280_i2c_0, offset 0x00)
Temp Register:0xFF200000 (BME280_REG_TEMP, 20-bit ADC → HPS compensates)
Calibration:Per-chip factory coefficients read via HPS /dev/i2c-1 at startup
Range:-40°C to +85°C
Accuracy:±0.5°C (calibrated) / ±1°C (uncalibrated)
Alarm Thresholds:Low 15°C / High 35°C (set in FPGA alarm_logic_0)
